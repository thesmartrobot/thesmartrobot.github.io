Drone radar dataset

This dataset contains camera and radar data from a drone flying in our lab.

The dataset consists of recording sequences categorized in a number of scenarios, each contained in hd5 files. In each file contains a hdf5 dataset structure labeled by sensor (i.e. radar and camera and depth). The radar is stored as raw ADC samples, and in addition we also provide post-processed range-doppler and range-azimuth maps. In addition, we also recorded the action commands (yaw, pitch, roll, thrust), and in scenarios where the motion capture system was active a ground truth 6DOF pose is provided. LatentSLAM dataset

This dataset contains camera, lidar and radar data recorded with a turtlebot2i in our lab.

The dataset consists of recording sequences categorized in 7 scenarios, each contained in hd5 files. In each file contains a hdf5 dataset structure labeled by sensor (lidar, radar and camera). Finally, the taken action is stored under the action key. Each scenario consists of a different route through the warehouse. The data can be downloaded by following this link
